System Requirements

The following Python packages must be installed:

  • Java 1.7
  • Maven. if in doubt, check https://github.com/bayolau/build_env/blob/master/maven3.3.9 for build and load examples
  • Python 2.7 (for usage examples)

Installing LongISLND

LongISLND can be built by running linux_build.sh, assuming wget, Java 1.7 (java) and Maven (mvn) are available. Please accept HDF5's license and accept default installation location to proceed.

Running LongISLND

Usage of the Java JAR can be carried out/demonstrated by the following convinient Python scripts.

simulate.py
  • simulate sequences given a model and a genome
sample.py
  • please refer to the Example below for input preparation.
  • build model used by simulate.py.
  • for PacBio's read, the script will look for files whose names are:
    • *.fofn, each storing a list of bax.h5 read files
    • *.fofn.cmp.h5, each storing the alignment of the corresponding *.fofn

Example

This is a toy demonstration of LongISLND, from downloading data, to alignment, to learning, and to simulation. Please execute in order.

Change working directory to sampling_example.

  • setup_smrt23.sh
    • download and build PacBio's SMRTAnalysis2.3
    • (to avoid complications) select NONE in answering the question "What job management system will you be using?"

Change working directory into sampling_example/ecoli.

  • download_and_align.sh
    • assume setup_smrt23.sh was executed in the directory sampling_example
    • download and align ecoli data
    • generate *.fofn and *.fofn.cmp5 used by sample.py
  • learn_and_simulate.sh
    • assume download_and_align.sh succeeded
    • learn model from aligned data
    • simulate reads and output in PacBio's BAM format
